# IMO-TSP

## requirements

 - Visual Studio 2022
 - CMake 3.15 and up
 - gnuplot (http://www.gnuplot.info/download.html)
 - matplot++ (https://github.com/alandefreitas/matplotplusplus/releases/tag/v1.2.0)

## How to run

 - open root project directory in VS22
 - run project (with main.exe as a startup item)
